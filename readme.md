# RactiveJS Experiment

My first attempt at SPA dev 3 years ago, be gentle!

### Setup

- make sure [node.js](http://nodejs.org), bower-installer, and [roots](http://roots.cx) are installed
- clone this repo down and `cd` into the folder
- run `npm install`
- run `bower install`
- run `roots watch`
- done
